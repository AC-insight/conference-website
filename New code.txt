        const lightbox = document.getElementById('lightbox');
        const lightboxImg = document.getElementById('lightbox-img');
        const lightboxCaption = document.getElementById('lightbox-caption');
        const galleryLinks = document.querySelectorAll('.gallery-link');
        const closeButton = document.querySelector('.close-button');

        const openLightbox = (link) => {
            const galleryItem = link.querySelector('.gallery-item');
            const caption = galleryItem.getAttribute('data-caption');
            
            lightboxImg.src = link.getAttribute('href');
            lightboxCaption.textContent = caption;
            lightbox.classList.add('is-visible');
            lightbox.setAttribute('aria-hidden', 'false');
            closeButton.focus(); // 自動聚焦到關閉按鈕
        };

        galleryLinks.forEach(link => {
            link.addEventListener('click', e => {
                e.preventDefault();
                openLightbox(link);
            });
        });

        const closeLightbox = () => {
            lightbox.classList.remove('is-visible');
            lightbox.setAttribute('aria-hidden', 'true');
        };

        closeButton.addEventListener('click', closeLightbox);
        lightbox.addEventListener('click', e => {
            if (e.target === lightbox) {
                closeLightbox();
            }
        });

        // 4. 新增：燈箱的鍵盤支援
        window.addEventListener('keydown', e => {
            if (e.key === 'Escape' && lightbox.classList.contains('is-visible')) {
                closeLightbox();
            }
        });
        // 讓關閉按鈕也能用 Enter 鍵觸發
        closeButton.addEventListener('keydown', e => {
            if (e.key === 'Enter') {
                closeLightbox();
            }
        });

### **第三步：你需要提供的進一步資訊**

這次的程式碼升級，已經將所有**不需你提供額外檔案**的建議都實作完畢了。現在，輪到你來完成剩下的部分，讓網站達到真正的完美。

**你的待辦清單 (To-Do List):**

1.  **更新聯絡信箱**：
    * **動作**：在我提供的 `index.html` 中，找到 `mailto:contact@ac-insight.org` 和 `contact@ac-insight.org` 這兩處，將它們換成你們聯盟**實際使用的聯絡信箱**。

2.  **提供網站圖示 (Favicon)**：
    * **說明**：Favicon 就是顯示在瀏覽器分頁上的那個小圖示，是網站品牌識別的重要一環。
    * **動作**：你需要請設計師提供一個正方形的、通常是 `16x16` 或 `32x32` 像素的 `.ico` 或 `.png` 格式的圖檔。將它命名為 `favicon.ico`，並**放置在你專案的根目錄下**（與 `index.html` 同層）。我已經在 `index.html` 的 `<head>` 區塊加入了載入它的程式碼。

3.  **優化圖片 (長期任務)**：
    * **說明**：為了讓網站載入得更快，特別是在網路速度較慢的地區，優化圖片是必要的。
    * **動作**：
        * **壓縮**：將你 `images` 資料夾中的所有照片，使用像 [TinyPNG](https://tinypng.com/) 這樣的線上工具進行壓縮，可以在不犧牲太多畫質的情況下，大幅減少檔案大小。
        * **格式**：考慮將圖片轉換為 `WebP` 這種更現代、壓縮率更高的格式。

4.  **提供社群分享用的代表圖**：
    * **說明**：當有人在 Facebook 或 Twitter 上分享你的網站連結時，會出現一張預覽圖。我們在 `index.html` 的 `<meta property="og:image"...>` 標籤中設定了這張圖的路徑。
    * **動作**：請確認 `images/conference-photo.png` 這張圖是你希望被用來當作社群分享預覽圖的。如果不是，請提供一張更具代表性的圖片（建議尺寸為 1200x630 像素），並更新 `index.html` 中的路徑。

完成以上這些由你主導的任務後，你的網站就真正達到了專業級的
